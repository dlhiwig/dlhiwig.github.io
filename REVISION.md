## Change Request Process

1. **Submit Request**: Open a GitHub Issue with the `change-request` label
2. **Review**: Maintainers will review the request within 5 business days
3. **Approval**: Approved changes will be scheduled for the next release
4. **Implementation**: Changes implemented and tested
5. **Release**: New version published with updated REVISION.md
